# LabAssist

LabAssist is a web application designed to automate laboratory processes and document flow. It saves great amount of time for laboratory workers. All they have to do for each research sample is to choose the applicant from list of applicants, select the sample and fill in some data form sampling report. After the analisys, they fill in the results and that's it. The test protocol, work logs, acts and protocols with prices for different applicants will be generated automatically on demand in .docx format. 

## What it does

* Maintains a database for all applicants, samples, test reports, etc.
* Generates test reports.
* Generates all the work logs.
* Generates acts and protocols for each applicant with price for all performed analysis from given date. 
* Application is secured with authentication policies with two levels of authority.
